# Python Practice Tasks

This repository contains two basic Python scripts for practice:

### Task 1: Dictionary of Student Marks
- Creates a dictionary with student names and their marks.
- Takes input for a student name.
- Displays the marks or a message if the student is not found.

### Task 2: List Slicing
- Creates a list of numbers from 1 to 10.
- Extracts the first five elements.
- Reverses and displays those elements.
